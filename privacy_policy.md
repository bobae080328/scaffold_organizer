# Privacy Policy (개인정보 처리방침)

Last updated: 2026-01-18

## 1. Overview

This application is an experimental productivity tool designed to process user-provided text input and generate structured task outputs.
The service is provided on an experimental, non-commercial basis.


## 2. Information We Collect

This application may process the following information only as provided directly by the user:
- Free-form text input (e.g., notes, task descriptions, or planning information)
- Optional configuration or preference inputs provided by the user
- No information is collected automatically, passively, or without user initiation.


## 3. Use of Information

User-provided information is used solely for the purpose of:
- Generating structured task lists or summaries
- Improving the immediate response quality within the same session
- The information is not used for profiling, advertising, or analytics purposes.

## 4. Data Storage and Retention

This application does not intentionally store personal data long-term.
Any data processed exists only temporarily for the purpose of generating responses, unless explicitly stored by the platform or the user.
The developer does not maintain an independent user database.


## 5. Data Sharing

User input may be transmitted to third-party AI service providers (e.g., language model APIs) solely to generate responses.
No data is sold, shared, or disclosed to third parties for commercial purposes.


## 6. User Responsibility

Users are advised not to submit sensitive personal data, including but not limited to:
- Medical information
- Financial information
- Government-issued identifiers
- Passwords or authentication credentials


## 7. Changes to This Policy

This Privacy Policy may be updated in the future as the application evolves.
Significant changes will be reflected by updating the “Last updated” date.


## 8. Contact

For questions regarding this Privacy Policy, please contact the developer through the platform on which this application is provided.
